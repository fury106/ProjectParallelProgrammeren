API¶

Package projectparallelprogrammeren¶

Een Monte Carlo simulatie van een aantal conformaties van een aantal atomen waarvan de Lennard Jones potentiaal berekend wordt.

class projectparallelprogrammeren.LCG(seed=None, a=2147483629, b=2147483629, m=2147483647)[source]¶

Linear congruential generator (LCG), gebaseerd op ET-rng.

Parameters

  • seed – Gebruikt standaard de huidige tijd om verschillende sequenties te berekenen.

  • a – Parameter voor de LCG.

  • b – Parameter voor de LCG.

  • m – Parameter voor de LCG.

UINT¶

alias of numpy.uint64

projectparallelprogrammeren.simulatie(n=5, m=10)[source]¶

Deze functie doet een simulatie van een gegeven hoeveelheid conformaties van het gegeven aantal atomen.

Parameters

  • n (int) – Het aantal atomen dat gebruikt wordt. (default 5)

  • m (int) – Het aantal conformaties dat gesimuleerd moet worden. (default 10)

Module projectparallelprogrammeren.atomen¶

Deze module bevat alle atomen die nodig zijn in de simulatie.

class projectparallelprogrammeren.atomen.Atomen(aantal)[source]¶

Deze klasse bevat enkel de coördinaten van de atomen, omdat er enkel de Lennard Jones potentiaal berekend zal worden.

afstandTweeAtomen(nummerAtoom1, nummerAtoom2)[source]¶

Deze functie berekent de afstand tussen de twee opgegeven atomen.

Parameters

  • nummerAtoom1 (int) – is het nummer van het eerste atoom.

  • nummerAtoom2 (int) – is het nummer van het tweede atoom.

berekenLJPot()[source]¶

Deze (test)functie berekent de totale LJ potentiaal van alle atomen. Van deze functie dient een Fortran of C++ variant gemaakt te worden, aangezien lussen traag zijn in Python.
